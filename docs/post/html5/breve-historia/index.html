Breve Historia

HTML fue creado y publicado por el físico Tim Berners-Lee en 1991 cuando se encontraba trabajando para el CERN (La Organización Europea para la Investigación Nuclear). Desarrollando sus labores cotidianas, Tim encontró una problemática, la cual era la forma en que se accedía y compartía las investigaciones realizadas.

Por lo cual, propuso un sistema de hipertexto basado en internet para compartir los documentos de una forma más eficiente y evitando las limitantes del programa que se usaba en el CERN, no solo definió el lenguaje, sino también el protocolo por el cual se comunicarían, el famoso protocolo HTTP (HyperText Transfer Protocol).

En un inicio HTML solo contaba con 18 etiquetas dentro de las cuales aún no figuraba los enlaces, pero serian agregados posteriormente. Existieron diferentes versiones de HTML, cada una de ellas con sus particularidades y mejoras.

  • HTML1
  • HTML2
  • HTML3
  • HTML4
  • HTML5 (el estándar oficial)

Sin embargo, HTML5 es una de las versiones de HTML con mayor auge y con un conjunto de mejores que permiten que los programadores seamos mejores en lo que hacemos.

En conclusión, recuerden que Tim Berners-Lee creo el famoso lenguaje de marcado de información HTML y el protocolo por el cual se comunicarían HTTP.
