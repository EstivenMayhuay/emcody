Breve Historia

HTML fue creado y publicado por el físico Tim Berners-Lee en 1991 cuando se encontraba trabajando en el CERN (La Organización Europea para la Investigación Nuclear).

Tim encontró una problemática, la cual era la forma en que se accedía y compartía la información acerca de las investigaciones en la organización.

Por lo cual, propuso un sistema de hipertexto basado en internet para compartir los documentos de una forma más eficiente y evitando las limitantes del programa que se usaba en el CERN, no solo definió el lenguaje sino también el protocolo por el cual se comunicarían (HTTP).

En un inicio HTML solo contaba con 18 etiquetas dentro de las cuales aún no figuraba los enlaces, pero serian agregados posteriormente. Existieron diferentes versiones de HTML como: HTML 1.0 HTML2, HTML 3, HTML 4 y el estándar oficial validado por la W3C que es HTML5.
